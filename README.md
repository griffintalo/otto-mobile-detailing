# Otto Mobile Detailing — Booking Site

Next.js + Supabase booking site for Otto Mobile Detailing.

## Setup

1. Copy `.env.local.example` to `.env.local` and fill in your values.
2. Run the SQL in `supabase-schema.sql` in your Supabase project's SQL Editor.
3. `npm install`
4. `npm run dev` to run locally, or deploy to Vercel with the same environment variables set in Project Settings.
